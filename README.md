# GMP, MPFR, Flint &amp; ARB for Windows

## Introduction

GMP, MPFR, Flint and ARB are numerical libraries for large integer and arbitrary precision arithmetic.

## Environment

- MSYS 1.0
- MSYS patch from 2012
- MinGW in short
- buildARB.sh

## Source

- GMP v5.1.3
- MPFR v3.1.2
- FLINT 2.4.5
- ARB v2.5.0+ (master branch head from 06.04.2015.)
